    What does it do?

    Splits an image into n equally-sized tiles. Also capable of joining the pieces back together.

    Whether you are planning a collaborative art project, creating a jigsaw puzzle, or simply get a kick out of dividing images into equally-sized quadrilaterals... this package is for you!

    Getting started

    Installation is as simple as:

        $ pip install image_slicer

    You can slice images from your Python code:

        >>> import image_slicer
    >>> file = 'cake.jpg'
    >>> num_tiles = 36
    >>> tiles = image_slicer.split_image(file, num_tiles)
    >>> image_slicer.save_tiles(tiles)

    ... or from the command line:

        $ slice-image cake.jpg 36
